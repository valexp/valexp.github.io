Protocol Features HSRP
Name Hot Standby Router Protocol
Scope Cisco Proprietary
Standard RFC 2281
OSI Layer Layer 3
Load Balancing No
Multicast Group IP Address 224.0.0.2 in v1
224.0.0.102 in v2
Transport Port Number UDP 1985
Timers Hello 3 seconds
Hold 10 seconds
Election Active Router:
1. Highest Priority
2. Highest IP
Router Role 1. One Active Router, One Standby Router
2. One or more Listening Routers
Group Virtual Mac Adress 0000.0C07.ACxx v1
0000.0C9F.Fxxx v2
Manual Virtual Mac Adress Assign Yes
Virtual IP can also be a real address No
Preempt enabled by default No
Multiple forwarding router per group No
Authentication supported Yes - MD5
Group ID v1: 0-255 (Default 0)
v2: 0-4095 (Default 0)
Priority 0-255 (Default 100)
IPv6 Support Yes (v2)
Comando Descrizione Modalità
STANDBY group IP virtual-ip Attivo HSRP assegnado il router ad un gruppo che dovrà essere uguale in tutti i router. Anche l'IP virtuale dovrà essere lo stesso e nella stessa subnet degli host Interface
STANDBY group PRIORITY priorità Opzionale,default 100. Quello con la priorità più alta diventa active. Interface
STANDBY group PREEMPT Opzionale, di default è disabilitato. Se abilitato il router che si collega per secondo,se ha una priorità più alta, diventerà active Interface
STANDBY VERSION numero Opzionale, di default viene usata la versione 1.Scelgo la versione da utilizzare: 1 o 2. Vale per tutti i gruppi Interface
